2. Using css to line through the content when checkbox checked.

[:checked](https://css-tricks.com/almanac/selectors/c/checked/)

```html
  <div class="item">
    <input type="checkbox" />
    <p>Don't forget to tweet your result!</p>
  </div>
```

```css
  input:checked + p {
    background: #f9f9f9;
    text-decoration: line-through;
  }
```
